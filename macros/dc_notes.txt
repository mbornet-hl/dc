Q : Comment faire un equivalent de : cmd && echo "OK" || echo "KO"


Creer une mecanique pour executer une macro :
	- si A && B  (AND)
	- si A || B  (OR)
	- si A et !B (XOR)

Creer une mecanique pour executer :
	- macro V si condition == VRAI
	- macro F si condition == FAUX

Q: comment executer une macro dont un connait le nom ?
R: si la macro est stockee dans Ra:
	lax
	=> pour executer une macro b :
	  stocker [lbx] dans a

	  Exemple :
	  ~~~~~~~
	  [[TOTO]P 10P]sb
	  [lbx]sa
	  lax
	  TOTO
